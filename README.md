# Eng. Software: Algoritmos e Lógica de Negócios

Repositório dedicado às atividades práticas de **Lógica de Programação com Python**, desenvolvidas durante o 1º período de **Engenharia de Software**.

O foco destes exercícios não é apenas a sintaxe da linguagem, mas a aplicação de **lógica de negócios** para resolver problemas reais de engenharia, logística e gestão.

## O que foi desenvolvido?

Este projeto contém 16 exercícios práticos resolvidos, simulando cenários reais:

### Engenharia & Indústria
- **Auditoria de Ativos:** Conversão de tempo de uso de robôs industriais para cálculo de depreciação.
- **Telemetria de Drones:** Cálculo de autonomia de voo e consumo energético (Joules) com margem de segurança.
- **Manutenção Preditiva:** Algoritmo para identificar ciclos de operação e paradas técnicas.

### Logística & Transporte
- **Otimização de Carga:** Algoritmo para calcular capacidade máxima de caminhões e evitar sobrepeso.
- **Gestão de Estoque:** Aplicação do fluxo IPO (Entrada-Processamento-Saída) para controle de inventário.

### Financeiro & RH
- **Folha de Pagamento:** Cálculo de horas trabalhadas vs. horas extras com taxas diferenciadas.
- **Engenharia Tributária:** Cálculo reverso de impostos sobre serviços e licenciamento de software.
- **Indicadores (KPIs):** Padronização de métricas para dashboards de diretoria.

## Tecnologias
- **Linguagem:** Python 3
- **Ferramentas:** Deepnote / Jupyter Notebook
- **Conceitos:** Tipagem forte, Operadores Matemáticos, Lógica Booleana, Manipulação de Strings.
